# Reports Artifacts

This folder stores generated report artifacts from model training and evaluation.

Current contents include ROC curves exported during training:

- `roc_logistic_regression.png`
- `roc_random_forest.png`

To refresh artifacts, run:

```bash
python -m src.train
```

For EDA charts, see `reports/figures/` generated by:

```bash
python -m src.eda --raw-data data/raw/heart_disease.csv --output-dir reports/figures
```
