# Screenshots Evidence

Use this folder for assignment evidence images.

Recommended files:

- `01_data_pipeline.png`: data download + preprocessing run output.
- `02_eda_outputs.png`: generated EDA figures (`reports/figures/*`).
- `03_mlflow_runs.png`: MLflow experiment table and run details.
- `04_ci_pipeline_success.png`: successful GitHub Actions run.
- `05_docker_predict.png`: local Docker prediction response.
- `06_k8s_resources.png`: `kubectl get pods,svc,ingress` output.
- `07_k8s_health_check.png`: `/health` and `/predict` verification.
- `08_metrics_endpoint.png`: `/metrics` output or dashboard view.

Tip: keep names stable so they can be referenced directly in `final_report.docx`.
